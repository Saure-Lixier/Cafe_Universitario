# ☕ Café Universitario

## Descripción

Sitio web desarrollado para el Certamen Unidad 2 de Programación Front End.

El proyecto simula el sitio web de una cafetería universitaria, incorporando HTML5, CSS3, JavaScript y control de versiones mediante Git y GitHub.

---

# Tecnologías utilizadas

## Desarrollo

- HTML5
- CSS3
- JavaScript

## Control de versiones

- Git
- GitHub

## Entorno de desarrollo

- Visual Studio Code

## Herramientas de apoyo

- Obsidian

## Inteligencia Artificial

- GitHub Copilot
- ChatGPT
- Claude AI

---

# Funcionalidades

- Navegación mediante menú principal.
- Banner principal con llamado a la acción.
- Sección institucional "Nosotros".
- Catálogo de productos dividido por categorías.
- Navegación interna entre categorías.
- Formulario de solicitud de información.
- Registro dinámico de solicitudes.
- Diseño responsive.
- Catálogo con imágenes y etiquetas informativas.
